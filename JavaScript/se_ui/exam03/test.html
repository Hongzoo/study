Q. 함수 호출시 new 를 붙인 것과 안 붙인 것의 차이를 설명하세요.

예시)
var obj = function () {
   console.log(this);
};
obj();
// new obj();
return 문이 없는 함수는 기본값을 반환합니다.
new 키워드로 호출되는 생성자의 경우, 기본값은 자신의 this 매개변수 값입니다.
다른 모든 함수의 경우, 기본 반환값은 undefined입니다.
new 키워드로 호출되는 생성자는 this가 함수 자신을 가리키고,
new 키워드 없이 호출되면 this가 윈도우 객체를 가리킵니다.
